## Quickstart

After completing the project

Open your terminal

```bash
  git add index.html
  git commit -m "[type a message]"
```

Push the commit

```bash
  git push [remote-name] main
```

- Go back to the original repo.
- Hit "new pull request" and compare between forks.
- Confirm the pull request and that's it!
